# Resume AI 2 Roadmap

## Version 0.1
- Desktop GUI with tabs.
- Candidate profile form.
- Job description input.
- Existing CV and resume input.
- Template selection.
- Generate tailored CV and resume as Markdown.
- Save profile locally.

## Version 0.2
- Add DOCX export.
- Add PDF export.
- Add stronger template layouts.
- Add validation warnings for missing dates, weak bullets, and unverifiable claims.

## Version 0.3
- Add AI settings tab.
- Add model selector.
- Add prompt preview.
- Add secure API key loading from environment variables.

## Version 0.4
- Add import support for PDF and DOCX.
- Add skill gap analysis against the job description.
- Add keyword matching score.

## Version 1.0
- Polished desktop app.
- Installer build.
- Automated tests.
- GitHub Actions workflow.
- Proper release notes.
